# gh0st_remote
gh0st远控复现，可通过文件夹包含的库选择是否调用制定目标进行远控的功能添加、删改。`（已停止维护）`

***本示例在实现时使用VS2015与VS2017开发，VS2017可能无法完全执行实现示例所有功能，且由于时间久远，在文件上传时未经过完整性检查，请参照文件内已命名为“2015Remote”的资源列表文件和导入项目文件自行进行检查***
